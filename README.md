
# README –  Final Project: Empower Ability Labs (SPA)

## **Overview**

Empower Ability Labs is a fully accessible **Single-Page Application (SPA)** built using HTML, CSS, and JavaScript. The project focuses on clean navigation, responsive design, and strong accessibility support following **WCAG 2.1 AA** and **WAI-ARIA Authoring Practices**.

The SPA contains three main views:

* **Home**
* **Services**
* **Schedule a Call**

All views are rendered within a single HTML file, with JavaScript handling smooth navigation and dynamic updates.

---

## **Features**

* Single-Page navigation with smooth view switching
* Responsive layout for desktop and mobile
* Semantic HTML structure with landmark elements
* Accessible modal dialog
* Custom switch component
* Expandable “Tell us about your event” area
* Accessible web form with validation and announcements
* Keyboard-friendly navigation throughout the interface

---

## **Accessibility**

The project incorporates:

* Proper focus management
* Skip-navigation link
* ARIA roles and attributes for custom components
* Screen reader friendly headings and labels
* Visible focus outlines
* High color contrast
* Keyboard-only usability
* Announced errors and success messages
